# Replace "Checking preview server…" text with Start Preview button

## What changed

In the desktop sidebar of the evolve session view, the plain text status message
"🔍 Checking preview server…" has been replaced with a large round play (▶) button
labelled **"Start Preview"** underneath it.

When the server is actively starting (after the button is clicked, or detected as
already booting), the button area changes to a pulsing circle with the text
"Starting preview…" to give clear feedback.

## Why

The old text was passive and gave users no obvious way to start the preview server
from the sidebar itself — they had to scroll down to the "Available Actions" panel
and find the small "▶ Start preview" link there. The new button is large, centred,
and immediately actionable, making the call-to-action obvious.
