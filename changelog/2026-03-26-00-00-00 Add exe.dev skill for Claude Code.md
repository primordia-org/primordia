# Add exe.dev skill for Claude Code

## What changed

Added `.claude/skills/using-exe-dev.md` — a Claude Code skill sourced from [https://github.com/boldsoftware/exe.dev/blob/main/skill/SKILL.md](https://github.com/boldsoftware/exe.dev/blob/main/skill/SKILL.md).

The skill file lives at `.claude/skills/using-exe-dev.md` and is automatically picked up by Claude Code when working in this repository.

## Why

Primordia already includes a `scripts/deploy-to-exe-dev.sh` script and documents the exe.dev deploy flow in PRIMORDIA.md. Adding this skill gives Claude Code contextual guidance whenever it encounters exe.dev infrastructure tasks — covering VM lifecycle commands (`ssh exe.dev new/ls/rm`), direct VM access (`ssh <vm>.exe.xyz`), file transfer, and important non-interactive SSH workarounds (e.g. `-o StrictHostKeyChecking=accept-new`) that are easy to get wrong in automated environments.
