# Sketch Primordia CLI core

Added a new documentation sketch for a future `primordia` CLI and package split that would move Primordia's core evolve, session, preview, branch, and deployment capabilities out of the Next.js app and optional reverse proxy.

The document outlines likely commands, framework-agnostic adapter contracts, a package split, local API mode, example workflows for terminal and Astro usage, and a migration plan for extracting existing internals into a reusable Primordia Core layer.
