# Use Codex favicon for Codex harness

Added a local Codex favicon asset and wired it into the shared agent identity icon map so the Codex harness renders the actual Codex mark instead of the fallback `CO` text badge.

The Codex harness label now normalizes from `codex` to `Codex`, and the model picker uses the same Codex icon for the `openai-codex` provider group for consistency.
