# Make evolve mode green

## What changed
Changed the evolve mode color scheme from amber/yellow to green throughout the UI:

- **ModeToggle**: The active "Evolve" button now highlights in green (`bg-green-700`) instead of amber.
- **Evolve mode banner**: The info bar shown when evolve mode is active is now green (`bg-green-900/40`, `border-green-700/50`, `text-green-300`).
- **Decision card** (related issues): The card and its links/buttons use green tones instead of amber.
- **Submit button**: The "Evolve" submit button is now green (`bg-green-600`) instead of amber.

## Why
User request via issue #63: "Make evolve mode green."
