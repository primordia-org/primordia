# Restore Claude logo color

The shared Claude logo component now uses the Claude mark's brand fill color instead of inheriting the surrounding text color.

The model picker's Anthropic provider icon now reuses that shared component instead of carrying its own copy of the SVG path, so Claude logo color stays consistent across billing sources, harness identity rows, presets, and model selection.

This restores the colored Claude logo now that Primordia displays many provider and harness icons side by side, where a monochrome Claude mark was harder to distinguish.
